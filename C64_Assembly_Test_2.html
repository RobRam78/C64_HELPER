C64 Assembly Language Test - Vol. 2

100 ALL NEW Questions: Beginner (1-40) | Intermediate (41-65) | Advanced (66-85) | Expert (86-100)

Beginner (Questions 1 - 40)
Q1. What does the NOP instruction do?
Correct Answer: Nothing (No Operation)
Q2. Which register is 16-bits wide in the 6510 CPU?
Correct Answer: Program Counter (PC)
Q3. How wide is the Stack Pointer (SP) on the 6502/6510?
Correct Answer: 8 bits
Q4. What is the total addressable memory space of the standard C64?
Correct Answer: 64 KB (65536 bytes)
Q5. Which condition triggers a BMI instruction?
Correct Answer: Negative flag is set
Q6. Which condition triggers a BPL instruction?
Correct Answer: Negative flag is clear
Q7. Which condition triggers a BCS instruction?
Correct Answer: Carry flag is set
Q8. Which condition triggers a BCC instruction?
Correct Answer: Carry flag is clear
Q9. What does the CLV instruction do?
Correct Answer: Clears the Overflow flag
Q10. What does RTI stand for?
Correct Answer: Return from Interrupt
Q11. What does RTS stand for?
Correct Answer: Return from Subroutine
Q12. What is the hexadecimal opcode byte for a BRK instruction?
Correct Answer: $00
Q13. What is the hexadecimal opcode byte for a NOP instruction?
Correct Answer: $EA
Q14. What does ADC stand for?
Correct Answer: Add with Carry
Q15. What does SBC stand for?
Correct Answer: Subtract with Carry
Q16. To perform an independent subtraction (SBC) accurately, what must you do first?
Correct Answer: Set the Carry flag (SEC)
Q17. What addressing mode is used in `JMP ($1000)`?
Correct Answer: Absolute Indirect
Q18. Can the JSR (Jump to Subroutine) instruction use indirect addressing natively?
Correct Answer: No, only Absolute addressing
Q19. What happens if you increment the X register (INX) when it holds the value $FF?
Correct Answer: It wraps around to $00
Q20. What is the standard hexadecimal value for the character 'Space' in PETSCII?
Correct Answer: $20
Q21. What does the instruction TAY do?
Correct Answer: Copies the Accumulator to the Y register
Q22. Which two status flags are automatically updated by TAY, TAX, INX, and DEY?
Correct Answer: Zero and Negative flags
Q23. Which bit in an 8-bit byte represents the Sign (Negative) bit in two's complement?
Correct Answer: Bit 7
Q24. What decimal value does the hexadecimal number $FF represent?
Correct Answer: 255
Q25. What hexadecimal value does the binary number %10101010 represent?
Correct Answer: $AA
Q26. In the standard C64 memory map, what resides between $D000 and $DFFF?
Correct Answer: Hardware I/O Registers (VIC, SID, CIA)
Q27. How large is the standard C64 Color RAM?
Correct Answer: 1000 bytes (1 KB)
Q28. How many columns and rows make up the standard C64 text screen?
Correct Answer: 40 columns, 25 rows
Q29. What is the default background color of the C64 screen on boot?
Correct Answer: Blue (Color 6)
Q30. What is the default border color of the C64 screen on boot?
Correct Answer: Light Blue (Color 14)
Q31. What memory address initiates the software reset sequence when called (SYS 64738)?
Correct Answer: $FCE2
Q32. How many clock cycles does a standard NOP instruction take to execute?
Correct Answer: 2 cycles
Q33. Which instruction sets the Decimal mode flag?
Correct Answer: SED
Q34. Which register determines the exact memory location for a Zero Page instruction like `LDA $10`?
Correct Answer: The instruction itself explicitly dictates it
Q35. What does the instruction `PHA` push onto the stack?
Correct Answer: The Accumulator
Q36. What is the correct syntax to load the Accumulator with the binary value %00001111?
Correct Answer: LDA #%00001111
Q37. Which instruction branch relies on the Overflow flag being set?
Correct Answer: BVS
Q38. Which instruction branch relies on the Overflow flag being clear?
Correct Answer: BVC
Q39. If the Carry flag is clear, what will `BCC $10` do?
Correct Answer: Branch forward to the specified label/address
Q40. What does the `CPX #$05` instruction do?
Correct Answer: Compares the X register to the literal value 5
Intermediate (Questions 41 - 65)
Q41. What Kernal ROM routine saves memory to a device (like a disk drive)?
Correct Answer: SAVE ($FFD8)
Q42. What Kernal ROM routine restores the default system I/O vectors?
Correct Answer: RESTOR ($FF8A)
Q43. What does the instruction `BIT $D011` do to the Negative flag?
Correct Answer: Sets it to match bit 7 of the value at $D011
Q44. Which index register must be used with the `LDA (zp,X)` addressing mode?
Correct Answer: X register
Q45. Which index register must be used with the `LDA (zp),Y` addressing mode?
Correct Answer: Y register
Q46. Can you push the X register directly to the stack with a single 6502 instruction?
Correct Answer: No, you must transfer it to A first (TXA, PHA)
Q47. What address holds the physical top of the 6502 hardware stack (the first byte pushed)?
Correct Answer: $01FF
Q48. What is the maximum distance a relative branch instruction (like BNE) can jump backward?
Correct Answer: 128 bytes
Q49. What is the maximum distance a relative branch instruction can jump forward?
Correct Answer: 127 bytes
Q50. How many bytes of memory does the instruction `LDA #$00` consume?
Correct Answer: 2 bytes
Q51. How many bytes of memory does the instruction `LDA $1234` consume?
Correct Answer: 3 bytes
Q52. How many bytes of memory does the instruction `TAX` consume?
Correct Answer: 1 byte
Q53. In what direction does the 6502 stack grow when items are pushed?
Correct Answer: Downwards (from $01FF towards $0100)
Q54. What happens to the Stack Pointer register when you push a byte onto the stack?
Correct Answer: It decrements by 1
Q55. Which hardware chip does CIA 1 ($DC00) interface with by default?
Correct Answer: Keyboard and Joysticks
Q56. Which hardware chip does CIA 2 ($DD00) interface with by default?
Correct Answer: Serial Bus (Disk Drive) and User Port
Q57. What register sets the data direction for CIA 1 Port A?
Correct Answer: $DC02
Q58. Keyboard matrix scanning relies on which CIA ports?
Correct Answer: CIA 1 Port A and Port B
Q59. Which type of hardware interrupt cannot be masked or disabled by the `SEI` instruction?
Correct Answer: NMI (Non-Maskable Interrupt)
Q60. What value is typically loaded into the Accumulator when you read `LDA $01` on a standard C64 boot?
Correct Answer: $37
Q61. The instruction `ASL A` shifts bits left. Where does the original bit 7 end up?
Correct Answer: In the Carry Flag
Q62. The instruction `LSR A` shifts bits right. Where does the original bit 0 end up?
Correct Answer: In the Carry Flag
Q63. What is the Kernal routine at $E544 typically used for?
Correct Answer: Clearing the screen
Q64. If the Carry flag is SET, what is the result of `ADC #$01` if the Accumulator holds $05?
Correct Answer: $07
Q65. If the Carry flag is CLEAR, what is the result of `SBC #$01` if the Accumulator holds $05?
Correct Answer: $03
Advanced (Questions 66 - 85)
Q66. Which VIC-II register is used to set a sprite into Multicolor mode?
Correct Answer: $D01C
Q67. Which VIC-II register expands a sprite to double its horizontal width?
Correct Answer: $D01D
Q68. Which VIC-II register expands a sprite to double its vertical height?
Correct Answer: $D017
Q69. Which SID register determines the waveform (Triangle, Sawtooth, Pulse, Noise) for Voice 1?
Correct Answer: $D404
Q70. What is 'Ring Modulation' in the context of the SID chip?
Correct Answer: Multiplying the output of one oscillator by another to create bell-like tones
Q71. Which CIA 2 port controls which 16KB memory bank the VIC-II can see?
Correct Answer: Port A ($DD00)
Q72. If CIA 2 Port A ($DD00) bits 0 and 1 are set to %11, which 16KB bank is the VIC-II viewing?
Correct Answer: Bank 0 ($0000 - $3FFF)
Q73. Where is the 6502 hardware NMI (Non-Maskable Interrupt) vector located?
Correct Answer: $FFFA / $FFFB
Q74. Where is the 6502 hardware RESET vector located?
Correct Answer: $FFFC / $FFFD
Q75. When writing a custom RAM IRQ handler while ROMs are active, which Kernal indirect vector is typically overwritten?
Correct Answer: $0314 / $0315
Q76. What action acknowledges (clears) a VIC-II Raster Interrupt request?
Correct Answer: Writing a 1 to bit 0 of $D019
Q77. What does clearing bit 4 of the VIC-II Screen Control Register 1 ($D011) achieve?
Correct Answer: It turns the screen blank, displaying only the border color
Q78. How many clock cycles does the instruction `INC $D019` take to execute?
Correct Answer: 6 cycles
Q79. What common instruction sequence is used in raster routines to create exact, cycle-precise delay loops?
Correct Answer: LDX #$xx / DEX / BNE
Q80. What is the pixel resolution of the standard C64 Hi-Res bitmap mode?
Correct Answer: 320 x 200
Q81. What is the pixel resolution of the C64 Multicolor bitmap mode?
Correct Answer: 160 x 200
Q82. In standard text mode, how many colors can be uniquely assigned to a single 8x8 character block?
Correct Answer: 2 (Background and Foreground)
Q83. In multicolor text mode, how many colors can be used within a single 4x8 character block?
Correct Answer: 4 colors
Q84. Which hardware register controls the master volume and filter selection of the SID chip?
Correct Answer: $D418
Q85. What happens if you execute an RTI instruction without having pushed a processor status and return address to the stack?
Correct Answer: The CPU pulls garbage data, usually leading to a crash
Expert (Questions 86 - 100)
Q86. What causes the dreaded 'VSP Bug' (Variable Screen Positioning bug) that can crash some C64s?
Correct Answer: DRAM refresh failure due to manipulated VIC-II memory fetch cycles
Q87. Exactly how many clock cycles are there in a standard PAL C64 raster line?
Correct Answer: 63
Q88. Exactly how many clock cycles are there in a standard NTSC C64 raster line?
Correct Answer: 65
Q89. Which illegal opcode performs a bitwise AND between the Accumulator and memory, then shifts the result right (LSR)?
Correct Answer: ALR (or ASR)
Q90. Which illegal opcode stores (A AND X) into memory, then subtracts an immediate value into X?
Correct Answer: SBX (or AXS)
Q91. Achieving an 'Open Border' in the horizontal direction explicitly depends on changing which register at a precise cycle?
Correct Answer: The 38/40 column bit in $D016
Q92. How many clock cycles does a single sprite fetch steal from the CPU on a specific raster line?
Correct Answer: 2 cycles
Q93. Without any DMA chip, how must a C64 coder quickly clear the entire 64KB of RAM?
Correct Answer: Using a highly unrolled CPU loop (e.g., repeating STA)
Q94. During which precise cycle range on a raster line does the VIC-II fetch data for Sprite 0?
Correct Answer: Cycles 55 - 56
Q95. Why is the immediate addressing mode of the illegal opcode `LAX #$xx` sometimes avoided in productions?
Correct Answer: It is highly unstable and dependent on manufacturing lines/temperature ('magic constant' noise)
Q96. In what exact state are the CPU flags immediately after a hardware power-on reset?
Correct Answer: Unknown/Random, except the Interrupt Disable (I) flag is set
Q97. What is the fundamental difference between the 6510's internal registers at $00 and $01?
Correct Answer: $00 is the Data Direction Register, $01 is the Data Port
Q98. If you completely bank out the Kernal ROM via $01, what must you do to ensure hardware interrupts don't crash the machine?
Correct Answer: Provide valid interrupt vectors in RAM at $FFFA-$FFFF
Q99. What does the highly specialized illegal opcode `SHX` (or `SYA`) do?
Correct Answer: Stores X ANDed with the high byte of the target address + 1
Q100. How can you trick the VIC-II into drawing no background/character graphics, displaying only the border color over the entire screen area?
Correct Answer: By clearing bit 4 of $D011 (Screen Blanking)
