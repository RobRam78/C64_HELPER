C64 Assembly Language Master Test

100 Questions: Beginner (1-40) | Intermediate (41-65) | Advanced (66-85) | Expert (86-100)

Beginner (Questions 1 - 40)
Q1. What does the LDA instruction do?
Correct Answer: Loads the Accumulator
Q2. Which register is primarily used for math operations?
Correct Answer: Accumulator (A)
Q3. What instruction stores the Accumulator into memory?
Correct Answer: STA
Q4. How wide are the registers (A, X, Y) in the C64's 6510 CPU?
Correct Answer: 8 bits
Q5. Which instruction copies the Accumulator to the X register?
Correct Answer: TAX
Q6. Which instruction copies the X register to the Accumulator?
Correct Answer: TXA
Q7. Which instruction copies the Accumulator to the Y register?
Correct Answer: TAY
Q8. Which instruction copies the Y register to the Accumulator?
Correct Answer: TYA
Q9. How do you increment the X register?
Correct Answer: INX
Q10. How do you decrement the Y register?
Correct Answer: DEY
Q11. Which instruction increments a value directly in memory?
Correct Answer: INC
Q12. Which instruction decrements a value directly in memory?
Correct Answer: DEC
Q13. What is the hexadecimal prefix commonly used in 6502 assembly?
Correct Answer: $
Q14. What is the binary prefix commonly used in 6502 assembly?
Correct Answer: %
Q15. How do you represent the immediate decimal value 10 in standard assembler syntax?
Correct Answer: #10
Q16. What does the instruction `LDA #$05` do?
Correct Answer: Loads the value 5 into A
Q17. What does the instruction `LDA $05` do?
Correct Answer: Loads the value at memory address $05 into A
Q18. Which instruction compares the Accumulator with a value?
Correct Answer: CMP
Q19. Which instruction compares the X register with a value?
Correct Answer: CPX
Q20. Which instruction compares the Y register with a value?
Correct Answer: CPY
Q21. What does BEQ stand for?
Correct Answer: Branch on Equal
Q22. What condition triggers a BNE instruction?
Correct Answer: Zero flag is clear (Not Equal)
Q23. What instruction unconditionally jumps to a new address?
Correct Answer: JMP
Q24. What instruction performs a bitwise AND on the Accumulator?
Correct Answer: AND
Q25. What instruction performs a bitwise OR on the Accumulator?
Correct Answer: ORA
Q26. What instruction performs a bitwise Exclusive-OR on the Accumulator?
Correct Answer: EOR
Q27. What address range is the Zero Page?
Correct Answer: $0000 - $00FF
Q28. Why is Zero Page memory special?
Correct Answer: It allows faster and shorter instructions
Q29. What address range is the Hardware Stack?
Correct Answer: $0100 - $01FF
Q30. What instruction pushes the Accumulator onto the stack?
Correct Answer: PHA
Q31. What instruction pulls (pops) a value from the stack into the Accumulator?
Correct Answer: PLA
Q32. What instruction jumps to a subroutine?
Correct Answer: JSR
Q33. What instruction returns from a subroutine?
Correct Answer: RTS
Q34. At what address does the C64 default Screen RAM start?
Correct Answer: $0400
Q35. At what address does the C64 default Color RAM start?
Correct Answer: $D800
Q36. At what address are the VIC-II graphics registers located?
Correct Answer: $D000
Q37. At what address are the SID sound registers located?
Correct Answer: $D400
Q38. At what address are the CIA 1 registers located?
Correct Answer: $DC00
Q39. At what address are the CIA 2 registers located?
Correct Answer: $DD00
Q40. What is the standard starting address for a BASIC program on the C64?
Correct Answer: $0801
Intermediate (Questions 41 - 65)
Q41. Which instruction clears the carry flag?
Correct Answer: CLC
Q42. Which instruction sets the carry flag?
Correct Answer: SEC
Q43. Before performing an addition (ADC), what flag must usually be cleared?
Correct Answer: Carry (C)
Q44. Before performing a subtraction (SBC), what flag must usually be set?
Correct Answer: Carry (C)
Q45. What addressing mode is used in `LDA #$FF`?
Correct Answer: Immediate
Q46. What addressing mode is used in `LDA $1234`?
Correct Answer: Absolute
Q47. What addressing mode is used in `LDA $10,X`?
Correct Answer: Zero Page Indexed
Q48. What addressing mode is used in `LDA $1234,X`?
Correct Answer: Absolute Indexed
Q49. What addressing mode is `LDA ($FB),Y`?
Correct Answer: Indirect Indexed
Q50. What addressing mode is `LDA ($FB,X)`?
Correct Answer: Indexed Indirect
Q51. Which instruction shifts the bits in the Accumulator left by one?
Correct Answer: ASL
Q52. Which instruction shifts the bits in the Accumulator right by one, shifting in a zero?
Correct Answer: LSR
Q53. Which instruction rotates the bits left, bringing the Carry flag into bit 0?
Correct Answer: ROL
Q54. Which instruction rotates the bits right, bringing the Carry flag into bit 7?
Correct Answer: ROR
Q55. What does the BIT instruction do?
Correct Answer: Tests bits in memory against the Accumulator without changing A
Q56. What Kernal ROM routine is typically used to output a character to the screen?
Correct Answer: CHROUT ($FFD2)
Q57. What Kernal ROM routine gets a character from the keyboard queue?
Correct Answer: GETIN ($FFE4)
Q58. What instruction pushes the Processor Status flags to the stack?
Correct Answer: PHP
Q59. What instruction pulls the Processor Status flags from the stack?
Correct Answer: PLP
Q60. Which instruction clears the Decimal mode flag?
Correct Answer: CLD
Q61. Which instruction sets the Interrupt Disable flag?
Correct Answer: SEI
Q62. Which instruction clears the Interrupt Disable flag, allowing IRQs?
Correct Answer: CLI
Q63. What happens if you use `BCC` after a `CMP`?
Correct Answer: Branches if A is less than the compared value
Q64. What instruction is used to return from an interrupt handler?
Correct Answer: RTI
Q65. Which vector points to the standard hardware IRQ routine on the C64?
Correct Answer: $0314 / $0315
Advanced (Questions 66 - 85)
Q66. What register at $D012 is crucial for smooth scrolling and raster effects?
Correct Answer: Raster Counter
Q67. How many hardware sprites does the VIC-II support natively?
Correct Answer: 8
Q68. To read raster lines beyond 255, which bit must be checked?
Correct Answer: Bit 7 of $D011
Q69. Which memory address controls the border color of the C64 screen?
Correct Answer: $D020
Q70. Which memory address controls the background color of the C64 screen?
Correct Answer: $D021
Q71. What is a 'Raster Interrupt'?
Correct Answer: An interrupt triggered when the screen draws a specific scanline
Q72. How do you acknowledge a VIC-II raster interrupt?
Correct Answer: INC $D019 (or store value to clear bit)
Q73. What does the SID chip's ADSR stand for?
Correct Answer: Attack, Decay, Sustain, Release
Q74. Which register configures the memory layout (RAM/ROM banking) on the C64?
Correct Answer: $01 (6510 IO Port)
Q75. To bank out BASIC ROM and access RAM underneath, what value is written to $01?
Correct Answer: $36 (or $35, etc.)
Q76. What register controls where the VIC-II looks for Screen RAM and Character Sets?
Correct Answer: $D018
Q77. What happens when the CPU encounters a BRK instruction ($00)?
Correct Answer: It triggers an IRQ/BRK interrupt sequence
Q78. Which vector does a BRK instruction jump through?
Correct Answer: $0316 / $0317 (or $FFFE)
Q79. What is a 'Bad Line' in VIC-II terminology?
Correct Answer: A scanline where the VIC-II halts the CPU to fetch character data
Q80. How many CPU cycles does a Bad Line steal?
Correct Answer: 40-43 cycles
Q81. What register controls the VIC-II memory bank selection (16KB blocks)?
Correct Answer: $DD00 (CIA 2 Port A)
Q82. How do you enable a specific sprite (e.g., Sprite 0)?
Correct Answer: Set bit 0 of $D015
Q83. Where do sprite pointers typically reside?
Correct Answer: The last 8 bytes of Screen RAM
Q84. What is the formula to calculate the address of a sprite's graphics data from its pointer?
Correct Answer: Pointer Value * 64
Q85. Which timer is typically used by the standard Kernal for the system clock and blinking cursor?
Correct Answer: CIA 1 Timer A
Expert (Questions 86 - 100)
Q86. What is an 'Illegal Opcode'?
Correct Answer: An undocumented instruction created by unintended silicon behavior
Q87. What does the illegal opcode `SAX` (also known as `AXS`) do?
Correct Answer: Stores A AND X into memory
Q88. What is the famous 'JMP ($xxFF) bug' on the original 6502?
Correct Answer: It fetches the high byte from $xx00 instead of $(xx+1)00
Q89. In precise cycle counting, how many clock cycles does a taken branch (same page) require?
Correct Answer: 3 cycles
Q90. How many clock cycles does a branch require if it crosses a page boundary?
Correct Answer: 4 cycles
Q91. What is 'VSP' (Variable Screen Positioning)?
Correct Answer: A trick that delays Bad Lines to shift the screen layout
Q92. What is 'FLD' (Flexible Line Distance)?
Correct Answer: Opening the vertical border by manipulating $D011
Q93. To open the upper and lower borders, what register must be changed at specific raster lines?
Correct Answer: $D011 (Screen Control 1)
Q94. To open the left and right side borders, what register must be changed?
Correct Answer: $D016 (Screen Control 2)
Q95. Which illegal opcode loads both the A and X registers simultaneously?
Correct Answer: LAX
Q96. Which illegal opcode decrements memory and compares it with the Accumulator?
Correct Answer: DCP
Q97. How do demo coders stabilize the raster to exact clock cycles?
Correct Answer: Double-IRQ synchronization
Q98. What is Self-Modifying Code?
Correct Answer: Code that overwrites its own instructions during execution
Q99. What is the cycle cost of `STA $1234,X` if it crosses a page boundary?
Correct Answer: 5 cycles (no penalty for write operations)
Q100. Which of the following is true about reading the VIC-II collision registers ($D01E, $D01F)?
Correct Answer: Reading them clears them automatically
